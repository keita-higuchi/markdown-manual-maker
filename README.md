# markdown-manual-maker

A tool to create PDF documents with Markdown.

## Usage

### How to split page

Split page by `---`.


### Change page settings

### Change page size

You can change the page size from the right top gear button.

### Change Layout

To change `text-align`,

`--!right`
`--!center`
`--!left`

with end tag `---!`

The flex layout is as follows.

```
---!flex
---!flexContent
content A
---!
---!flexContent
content B
---!
---!
```
